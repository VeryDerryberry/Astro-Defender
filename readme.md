Game Title: Astro Defender
Core Gameplay Loop:
You control a fast, agile spaceship in a 2D arena. Enemies continuously spawn from the edges of the screen and fly toward you. Your job is to survive as long as possible by destroying enemies while avoiding collisions.
Controls:

Arrow keys or WASD: Move/Thrust
Mouse: Aim
Left Click: Shoot

Game Flow:

Start Screen with "Press Any Key to Start"
Main gameplay with increasing difficulty (more and faster enemies over time)
Simple lives system (start with 3 lives)
Game Over screen showing final score
Local high score saved

Style: Clean retro vector graphics using only Line2D and Polygon2D nodes. No sprite images.
